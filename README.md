## Описание

Training Planner — удобное приложение с графическим интерфейсом на Python (Tkinter), предназначенное для планирования и учёта тренировок. Программа позволяет добавлять новые тренировки, указывая дату, тип занятия и его длительность, а также просматривать все сохранённые записи в таблице.

Для удобства пользователя реализована система фильтрации по типу тренировки и диапазону дат, что помогает быстро находить нужные занятия. Все данные сохраняются в формате JSON и автоматически загружаются при следующем запуске программы.

Приложение имеет простой и понятный интерфейс, подходит для личного использования и помогает контролировать регулярность тренировок.
